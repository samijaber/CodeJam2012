Codejam 2012
=======

Description
===========

An app that I wrote with 2 teammates during a 48h hackathon at McGill (CodeJam @ McGill). 


To run the app:

1- run the server by writing out the command found in /exchange/command.txt

2- run "node app.js"

3- open your browser to http://0.0.0.0:8888/
